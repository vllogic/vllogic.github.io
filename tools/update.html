Vllink HID Update

Firmware Update

Readme

1. 下载固件

2. 先按住设备上的按键，再连接电脑

3. 选中弹框中的"Vllink xxx Update"，点击连接

4. 在"Hardware Info"栏目中会显示硬件序列号

5. 在"Software Version"栏目中会显示设备已有固件版本

6. 点击"选择文件"，打开需要刷入的固件

7. 点击"Update"，稍待片刻，固件更新完成

8. "Auto Reset"选项默认选中，在固件更新完成后，设备会自动重启，进入一般模式

9. 一般模式下，本升级固件无法与设备通讯
