常规设置

Basic2配置方法:

1. 点击“Connect Vllink”

2. 在小弹窗中选中DAP，点击连接

3. 连接后，上面的常规设置文本即可修改，修改后需要点击“同步设置”保存，注意无效的配置将在同步时被清除

配置文本说明：

1. Mode：指定调试器模式。其中，'wired'对应有线模式；'wirelesshost'对应AP即无线主机模式；'wirelessdevice'对应STA即无线设备模式

2. Mode_Quick_Switch：双击按键快速切换模式功能。其中，'enable'对应开启；'disable'对应关闭

3. Vref_Voltage_mV：指定VRef模式。其中，'0'对应输入模式；'1800'对应输出1.8V；'2500'对应输出2.5V；'3300'对应输出3.3V；'4000'对应输出4.0V

4. Wireless_Device_SSID：指定配对AP SSID。已配对的STA会显示对应AP的SSID，可通过删除'='之后的文本让STA重新配对AP

5. Wireless_Device_Password：指定配对AP Password。已配对的STA会显示对应AP的Password

6. Wireless_Host_Channel：指定AP信道。其中，'165'为推荐信道，其他可选信道有36, 40, 44, 48, 149, 153, 157, 161
