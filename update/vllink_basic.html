Vllink Basic 固件更新

更新方法

  1. 解压压缩包

  2. 按住按键连接Win10电脑，再使用Chrome浏览器打开更新页面 WebDFU@Github或WebDFU@Gitee

  3. 点击Connect，找到Vllink Basic DFU Bootloader并双击

  4. 在Firmware Download栏目中选中解压得到.bin文件

  5. 点击Download，等待一段时间后，完成更新。
